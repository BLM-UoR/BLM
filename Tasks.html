1. Assignment 1 ActivitiesÂ¶

These are the various topics to be addressed in Assignment 1. They are designed to be in a logical calculation order to help you understand different physical properties and processes.

Tip

The tasks designed here are based GitHub using Python 3. We recommend these softwares/bundles (with download links) as tools for finishing tasks:

  1. Anaconda 3: an easy-to-use Python 3 bundle with many preinstalled data analysis packages. download Anaconda 3

  2. GitHub Desktop: the official git client by GitHub with essential git functionalities. download GitHub Desktop

Activities

  • 1.1. Understanding Surface Parameters (1): Albedo
  • 1.2. Visit to Observatory
  • 1.3. Understanding Surface Properties (2): LAI
  • 1.4. Understanding Surface Properties (3): Surface Roughness
  • 1.5. Modelling \(\Delta Q_S\) with OHM
  • 1.6. Understanding Surface Properties (4): Surface Resistance
  • 1.7. Modelling \(Q_E\) and \(Q_H\) with Penman-Monteith
  • 1.8. MT49E Extra: Sensitivity Analysis of Modelled \(Q_E\)

Note

  1. When you write your paper, the presentation will be as a coherent scientific paper, not a set of individual tasks, not necessarily in this order and not including each of the individual components (e.g. Week 1 plots are intended to get you going).

  2. These tasks are to be undertaken in assigned groups according to land cover types, but independent papers will be written (without plagarism).
